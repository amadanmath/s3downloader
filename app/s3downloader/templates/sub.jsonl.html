For your record, you can download the attached JSONL file. The file is encrypted; to view it, you will need to decrypt it using your private key:

openssl smime -decrypt -binary -inform DER -inkey "$PRIVATE_KEY" -in {{attachment_name}}

Alternately, you can use the script bin/decrypt.sh like this:

bin/decrypt.sh -k "$PRIVATE_KEY" {{prefix}}-{{corpus_name}}-*.jsonl.enc
